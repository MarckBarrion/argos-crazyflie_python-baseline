# Compiling the code

Make sure you have ARGoS >= 3.0.0-beta52 installed!

Commands:

```shell
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
make
```

## Example
```shell
argos3 -c src/experiments/crazyflie_battery.argos
```

## Example
```shell
argos3 -c src/experiments/crazyflie_position.argos
```
